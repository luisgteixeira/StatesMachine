States Machine Basic Operations
==

Input file format:

<pre>
  <code>
  q1,q2,q3     -> States separated by coma
  a,b          -> Events separated by coma
  q1           -> Initial state
  q2,q3        -> Final states separated by coma
  q1-a-q2      -> States transitions. One by line.
  q2-b-q3
  </code>
</pre>
